# Spring Boot Backend Skill

## Responsibilities

- Create Controllers, Services, Repositories
- Implement REST APIs
- Use JPA/Hibernate
- Follow layered architecture

## Rules

- Use @RestController
- Use DTOs for request/response
- Use Service layer for business logic
- Use Repository for DB access
- Add validation using @Valid

## Output Format

- Controller class
- Service class
- Repository interface
- Entity class
- DTO classes

## Example Trigger

- "Create employee API"
- "Implement attendance module"
